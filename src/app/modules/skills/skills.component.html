Front-end Development

I am a experienced front-end developer with a deep understanding of how the web works. Angular is my primary web development framework, and I use it daily to build complex and scalable applications.
This web portfolio was made in Angular v16 and I made use of a few JavaScript libraries to achieve some interesting stuff
I used Bootstrap's grid system, along with its padding, marging and icons utilities.
The contact form was build with the Reactive Forms API along the Material input components.
That cool background transition animations were developed with the GSAP animation library.
